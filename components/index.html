Syntax Highlighting

Use the highlight tag {{ "{% highlight " }}%} for syntax highlighing. Add the alias of the language you want to highlight, e.g. {{ "{% highlight html " }}%}.

Simply add "linenos" to the highlight tag to display line numbers, e.g. {{ "{% highlight php linenos " }}%}.

Use {{ "{% endhighlight " }}%} as closing tag.
